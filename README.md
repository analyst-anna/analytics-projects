## 🎮 Интерактивность и Навигация

Дашборд спроектирован как **полностью интерактивный инструмент**, позволяющий пользователю самостоятельно исследовать данные без необходимости запрашивать новые отчеты.

###  Гибкая фильтрация по времени
Главным элементом управления является **срез по датам**, расположенный в шапке отчета.
- **Выбор периода:** Пользователь может задать любой произвольный диапазон дат (например, `01.01.2022` — `30.09.2022`).
- **Динамический пересчет:** При изменении дат в срезе **все** визуализации на странице (карточки KPI, графики динамики, таблицы топов) мгновенно пересчитываются под выбранный период.
- **Сравнение периодов:** Позволяет оперативно оценить сезонность и тренды, переключаясь между месяцами, кварталами или неделями.

###  Глубокая кросс-фильтрация
Все визуальные элементы связаны между собой. Клик по любому объекту на дашборде фильтрует остальные данные:

1.  **Анализ по Регионам:**
    *   При нажатии на столбец конкретного региона (например, *«Москва»*) в графике **«Продажи по областям»**:
    *  График динамики ROI покажет рентабельность **только по Москве**.
    *   Таблица **«Лидеры роста»** отсортирует товары, которые лучше всего продавались именно в этом регионе.

2.  **Детализация по Артикулу (SKU):**
    *   При клике на конкретный товар (например, *`supplierArticle 96`*) в таблице **«Лидеры возвратов»**:
    *   Карточка **«Выручка WB»** покажет доход только по этому артикулу.
    *   График **«Динамика выручки»** отобразит продажи данного товара во времени.

3.  **Изоляция данных:**
    *   При наведении курсора на точку графика подсвечиваются соответствующие данные в таблицах, что позволяет быстро сопоставить визуальный тренд с точными цифрами.

### 💡 Сценарии использования
*   **Сценарий «Почему упала прибыль?»**
    - Пользователь выбирает проблемный месяц в срезе дат.
    - Смотрит таблицу «Лидеры падения» — находит товар с минусовой динамикой.
    - Кликает на этот товар в таблице — видит, что в конкретном регионе (например, *«Свердловская обл.»*) были аномальные возвраты.
*   **Сценарий «Оценка эффективности региона»**
    - Кликает на *«Санкт-Петербург»* в графике продаж.
    - Анализирует ROI по этому региону.
    - Принимает решение об увеличении стока на местном складе.

### Блок 2: Динамика

**Динамика выручки**
- Линейный график по дням (янв 2022 - сен 2022)
- Ось X: Период (дни)
- Ось Y: Выручка (млн ₽)
- Показывает сезонные колебания и тренды продаж

**Динамика ROI**
- Линейный график с показателем рентабельности инвестиций
- Ось X: Период (дни)
- Ось Y: ROI (%)
- Отображает эффективность инвестиций в товарные запасы

### Блок 3: Аналитика

**Продажи по областям**
- Горизонтальная гистограмма
- Топ регионов:
  - Москва
  - Московская область
  - Санкт-Петербург
- Показывает географическое распределение выручки

**Лидеры роста**
- Таблица с артикулами по приросту выручки относительно аналогичного прошлого периода
- Колонки:
  - Артикул
  - Выручка WB (текущий период)
  - Изменение выручки (абсолютное)
  - Изменение выручки (%)
- Сортировка: по убыванию прироста

**Лидеры падения**
- Таблица с артикулами, показавшими отрицательную динамику относительно аналогичного прошлого периода
- Колонки:
  - Артикул
  - Выручка WB (текущий период)
  - Изменение выручки (абсолютное)
  - Изменение выручки (%)
- Сортировка: по возрастанию (наибольшее падение сверху)

**Лидеры возвратов**
- Топ товаров по количеству возвратов
- Колонки:
  - Артикул
  - Количество возвратов (шт)
- Помогает выявить проблемные позиции

##  Бизнес-инсайты

###  Выявлено в ходе анализа:

1. **Высокая рентабельность** — 59% указывает на эффективную бизнес-модель
   - Валовая прибыль составляет более половины выручки
   - Бизнес генерирует стабильный денежный поток

2. **Географическая концентрация** — основная выручка приходится на Москву и МО
   - Центральный федеральный округ доминирует в продажах
   - Есть потенциал для расширения в другие регионы

3. **Неравномерная динамика** — наблюдаются сезонные пики и спады
   - Периоды высокой активности чередуются с низкими продажами
   - Требуется планирование запасов с учётом сезонности

4. **Товарная дифференциация** — некоторые артикулы показывают рост >1000%, другие падают
   - Поляризация ассортимента по эффективности
   - Необходима оптимизация товарной матрицы

## 💡 Рекомендации

### 1. Оптимизация логистики
**Пересмотреть приоритетные склады с учётом географии продаж**

**Действия:**
- Увеличить запасы на складах Москвы и МО (основной спрос)
- Проанализировать целесообразность складов в регионах с низким спросом
- Оптимизировать маршруты доставки для снижения логистических затрат

**Ожидаемый эффект:**
- Снижение затрат на логистику на 10-15%
- Ускорение доставки до клиентов

---

### 2. Ассортиментная политика
**Сфокусироваться на товарах-лидерах роста**

**Действия:**
- Увеличить закупки артикулов с ростом >500%
- Вывести из ассортимента товары с устойчивым падением
- Провести ABC-анализ ассортимента

**Ожидаемый эффект:**
- Рост оборачиваемости товаров
- Увеличение общей выручки на 15-20%

---

### 3. Работа с возвратами
**Проанализировать причины возвратов топ-артикулов**

**Действия:**
- Выявить топ-10 товаров по количеству возвратов
- Проанализировать причины (брак, несоответствие описанию, размер)
- Улучшить карточки товаров (фото, описание, размерная сетка)
- Ввести контроль качества перед отгрузкой

**Ожидаемый эффект:**
- Снижение процента возвратов на 20-30%
- Экономия на логистике возвратов
- Повышение рейтинга продавца на WB

---

### 4. Ценообразование
**Учесть реальную себестоимость с логистикой при установке цен**

**Действия:**
- Рассчитать полную себестоимость для каждого артикула:
  - Закупочная цена
  - Комиссия WB (12%)
  - Логистика (55-220 ₽/ед. в зависимости от категории)
  - Хранение (0.08 ₽/день)
  - Налог
- Установить минимальную цену с целевой рентабельностью 40-50%
- Внедрить динамическое ценообразование для товаров с низкой маржой

**Ожидаемый эффект:**
- Исключение убыточных продаж
- Стабилизация рентабельности на уровне 50-60%
