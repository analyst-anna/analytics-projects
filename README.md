# Проект 5: Анализ пролонгации договоров (2023)

##  О проекте

**Заказчик:** Руководитель отдела сопровождения клиентов  
**Цель:** Оценить эффективность работы аккаунт-менеджеров по пролонгации договоров с клиентами в 2023 году
**Формат результата:** Аналитический отчёт в формате **Microsoft Excel** (`.xlsx`) с коэффициентами пролонгации, визуализациями и бизнес-рекомендациями

**Ключевые вопросы:**
1. Насколько хорошо менеджеры пролонгируют договоры в первый и второй месяц?
2. Кто из менеджеров показывает лучшие/худшие результаты?
3. Какие месяцы наиболее успешны для пролонгации?
4. Какие рекомендации можно дать для улучшения показателей?

##  Методология расчёта

### Коэффициенты пролонгации

| Коэффициент | Описание | Формула |
|-------------|----------|---------|
| **K1** | Пролонгация в первый месяц | Σ(отгрузка в M+1) / Σ(отгрузка в M) |
| **K2** | Пролонгация во второй месяц | Σ(отгрузка в M+2) / Σ(отгрузка в M-1) для проектов без отгрузки в M |

### Пример расчёта для мая 2023

| Коэффициент | Формула | Расшифровка |
|-------------|---------|-------------|
| **K1 (май)** | Σ(отгрузка в мае за проекты, завершившиеся в апреле) / Σ(отгрузка в апреле за те же проекты) | Пролонгация в первый месяц после завершения |
| **K2 (май)** | Σ(отгрузка в мае за проекты, завершившиеся в марте, без отгрузки в апреле) / Σ(отгрузка в марте за те же проекты) | Пролонгация во второй месяц для проектов, не пролонгированных в первый |

## Бизнес-рекомендации

### 🔴 Высокий приоритет (1-2 недели)

| Рекомендация | Обоснование |
|--------------|-------------|
| **Внедрить систему напоминаний о пролонгации** | За 60 и 30 дней до завершения договора — снизит количество «нулевых» месяцев |
| **Провести обучение для аутсайдеров** | Васильев А.А. (1,2%) требует срочной дополнительной подготовки |

### 🟡 Средний приоритет (1 месяц)

| Рекомендация | Обоснование |
|--------------|-------------|
| **Разработать скрипты работы с клиентами** | Для повышения K2 (сейчас 2,26% годовых) — внедрить повторные напоминания |
| **Создать KPI по пролонгации** | Целевой уровень K1 — 30% для всех менеджеров |

### 🟢 Низкий приоритет (квартал)

| Рекомендация | Обоснование |
|--------------|-------------|
| **Наградить лидеров** | Петрова А.Д. (111%) — премия за выдающийся результат |
| **Проанализировать февральский и октябрьский пики** | K1 = 64% и 57% — выявить факторы успеха и тиражировать на другие месяцы |

##  Анализ выбросов и аномалий

### Выявленные аномалии

1. **Февраль и октябрь — пики K1 (64% и 57%)**
   - Аномально высокие показатели на фоне остальных месяцев
   - Рекомендуется детально проанализировать причины

2. **Март–июнь — нулевой K1**
   - Четыре месяца подряд без единой пролонгации
   - Системная проблема, требующая немедленного вмешательства

3. **K2 = 0% в 8 месяцах из 12**
   - Клиенты либо продлевают сразу, либо не возвращаются
   - Необходима стратегия для удержания отложенных решений

4. **Поляризация менеджеров**
   - Лидеры (Петрова, Михайлов, Кузнецов, Смирнова) показывают >30%
   - Аутсайдер (Васильев) — 1,2%
